Teacher notes

You can replace the sample words with your own course vocabulary, workplace English items, or minimal pairs. Each card is stored in a simple JavaScript list inside the file.

Use in class

Project the page, click the model pronunciation, and ask students to repeat as a whole class, in pairs, or during station rotation.

Change the content

Edit the word, IPA, sentence, and category values in the script. You can build sets for vowels, consonants, stress, or topic vocabulary.

Add real recordings

Later, you can replace browser speech with your own MP3 files and keep the same button layout for GitHub Pages.

This starter site uses browser speech synthesis, so the exact voice depends on the learner’s device and browser. For more controlled models, you can later swap in recorded audio files or TTS exports.
